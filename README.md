# super-store-sales-excel-dashboard
Sales performance dashboard analyzing 9,994 retail records using Excel and Power Query

## Project Overview
Analyzed 9,994 retail sales records to evaluate revenue trends, regional performance, customer segments, and shipping modes using Microsoft Excel and Power Query.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Slicers & Charts

## Key KPIs
- Total Sales and Average Sales
- Total Quantity
- Total Transactions
- Regional Revenue Distribution
- Year Wise Analysis

## Key Insights
- Identified top-performing regions contributing the highest revenue.
- Analyzed shipping modes to evaluate cost-heavy segments.
- Evaluated customer segments to highlight high-value contributors.
- Enabled dynamic filtering using slicers for comparative performance analysis.

## Dataset Size
9,994 records
